# FiveM Installation on Linux/Ubuntu

This guide explains how to install a FiveM server on a Linux/Ubuntu system, step by step.

## Creating the FiveM User

:::note
For security reasons, it's recommended to create a dedicated user for your FiveM server rather than using the root user.
:::

## Connect to your server via SSH

```bash title="[command] - Create a new user named 'fivem'"
sudo adduser fivem
```

:::info
The system will ask you to set a password for the `fivem` user. 
Choose a secure password and make sure to note it down!
:::

```bash title="[command] - Add the user to the sudo group for administrative permissions"
sudo usermod -aG sudo fivem
```

```bash title="[command] - Log in as the new user"
su - fivem
```

:::tip title="Visual demonstration"
![Creating a FiveM user](/img/tutorial-fivem/create-fivem-user.gif)
:::

or reconnect via SSH with this new user

## Installing Dependencies

Create an installation script for the dependencies required by FiveM.

### Step 1: Create the Installation Script

```bash title="[command] - Create a new file for the script "
nano install_dependencies.sh
```

### Step 2: Copy the Following Code into the File

```bash title="install_dependencies.sh"
#!/bin/bash
set -e
# Colors
RED='\033[0;31m'
GREEN='\033[0;32m'
YELLOW='\033[1;33m'
BLUE='\033[0;34m'
NC='\033[0m'
log_info() { echo -e "${BLUE}[INFO]${NC} $1"; }
log_success() { echo -e "${GREEN}[OK]${NC} $1"; }
log_warning() { echo -e "${YELLOW}[WARNING]${NC} $1"; }
log_error() { echo -e "${RED}[ERROR]${NC} $1"; }
echo "FIVEM DEPENDENCIES INSTALLATION"
echo "=============================="
echo "Installing prerequisites..."
# System update
log_info "Updating APT sources"
if ! sudo apt update; then
    log_error "Failed to update sources"
    exit 1
fi
log_success "Sources updated"
log_info "Updating system (may take time...)"
if ! sudo apt upgrade -y; then
    log_warning "System update issue (continuing)"
else
    log_success "System updated"
fi
echo ""
# Dependencies installation
log_info "Installing dependencies"
dependencies=("wget" "xz-utils" "git" "nano" "curl" "unzip" "screen" "jq" "rsync")
for dep in "${dependencies[@]}"; do
    if ! command -v "$dep" &> /dev/null; then
        log_info "Installing $dep"
        if ! sudo apt install -y "$dep"; then
            log_error "Failed to install $dep"
            exit 1
        fi
    else
        log_success "$dep already installed"
    fi
done
echo ""
log_success "Dependencies installed!"
```

### Step 3: Make the Script Executable and Run It

```bash title="[command] - Make the script executable"
chmod +x install_dependencies.sh
```

```bash title="[command] - Run the script"
./install_dependencies.sh
```

:::tip
This script installs all the necessary dependencies to run a FiveM server on Linux, including basic tools like wget, git, curl, and screen which are essential for server management.
:::

:::note
If you encounter permission errors when running the script, make sure your user has sudo rights or run the command with `sudo` in front.
:::

## What Does This Script Do?

The installation script performs the following operations:

1. **System Update** - Updates APT sources and system packages
2. **Dependencies Installation** - Installs all packages necessary for FiveM to function:
   - wget: For downloading files
   - xz-utils: For decompressing archives
   - git: For version control
   - nano: Simple text editor
   - curl: Data transfer tool
   - unzip: For decompressing archives
   - screen: For running processes in the background
   - jq: For manipulating JSON data
   - rsync: For file synchronization

In the next sections, we'll see how to download and configure the FiveM server itself.
